# Second Life Project

- 프로젝트 명: <b>Second Life</b>
- 프로젝트 목표: 의류 재사용과 환경 보호를 위한 <b>중고의류 거래</b> 쇼핑몰 제작
- 팀원 구성 및 이름
  - 기획 & 디자인 : 이진희, 최연정
  - 프론트 : <b>민지원, 이경은, 추홍규</b>
  - 백앤드 : 김대희
- 배포 페이지: https://mapo-project.github.io/SecondLife-frontend/
