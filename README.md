# Second Life Project

- 프로젝트 명: **Second Life**
- 프로젝트 목표: 의류 재사용과 환경 보호를 위한 **중고의류 거래 쇼핑몰** 제작

- 팀원 구성 및 이름
  - 기획 & 디자인 : 이진희, 최연정
