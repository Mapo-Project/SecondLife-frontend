- React를 활용한 헌 옷 수거 플랫폼 "세컨드 라이프"

## 주요 기능

- 로그인
  ```
  - JWT 방식
  - refreshToken 쿠키에 저장
  - accessToken ReduxToolkit store에 저장
  - 간편 로그인: 카카오, 구글 활용
  ```
- 회원가입
  ```
  - useForm 훅 활용
  -
  - 간편 회원가입시 프로필 추가
  ```
